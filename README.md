ระบบจดจำใบหน้า (Face Recognition System) ที่พัฒนาด้วย OpenCV Python  
ออกแบบมาเพื่อใช้งานภายในห้องปฏิบัติการของคณะวิศวกรรมศาสตร์  
มหาวิทยาลัยธรรมศาสตร์ โดยมีเป้าหมายเพื่อช่วยตรวจสอบตัวตนผู้ใช้งาน  
และเพิ่มความปลอดภัยในการเข้าใช้งานพื้นที่เฉพาะ

โปรเจกต์นี้ยังอยู่ระหว่างการพัฒนา (Under Development)  
โดยปัจจุบันกำลังพัฒนาระบบประมวลผลภาพ, จัดเก็บข้อมูลผู้ใช้งาน  
และปรับปรุงประสิทธิภาพของโมเดลจดจำใบหน้าให้มีความแม่นยำมากขึ้น
