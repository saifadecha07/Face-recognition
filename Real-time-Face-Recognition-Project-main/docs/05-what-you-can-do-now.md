# What You Can Do Now

## ตอนนี้โปรเจกต์ของคุณพร้อมสำหรับอะไรบ้าง

1. ใช้กล้องจริงหรือ RTSP stream เพื่อตรวจจับใบหน้าแบบต่อเนื่อง
2. ระบุคนที่รู้จักจาก dataset ที่ถ่ายเก็บไว้
3. แยกคนไม่รู้จักเป็น `Unknown`
4. รู้ว่าคนนั้นยังอยู่ในกล้องหรือออกจากกล้องแล้ว
5. ตรวจได้ว่าคนนั้นมีการเคลื่อนไหวจากตำแหน่งในเฟรม
6. เก็บข้อมูลลง PostgreSQL เพื่อดูย้อนหลังได้
7. เปิด dashboard ให้ทีมงานดูสถานะได้แบบสด ๆ
8. deploy ระบบทั้งชุดบน server ตัวเองด้วย Docker

## ใช้ต่อเชิงงานจริงได้แบบไหน

- ระบบดูคนเข้าออกห้อง
- ระบบบันทึก attendance
- ระบบ monitor จุดทางเข้า
- ระบบเฝ้าระวังคนไม่รู้จัก
- ระบบหลายกล้องในอนาคต โดยแยก `camera_name`
- ระบบเชื่อม webhook หรือ mobile app ต่อจาก event ในฐานข้อมูล

## ถ้าจะพัฒนาเฟสต่อไป

- ทำ web dashboard แบบมีภาพ snapshot
- เปลี่ยน model recognition ให้แม่นขึ้น
- เพิ่มสิทธิ์ผู้ใช้และ login
- เพิ่มค้นหาย้อนหลังตามชื่อและเวลา
- เพิ่ม export report
